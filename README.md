# Alsimah
.منصة رقمية شاملة تعمل كهوية رقمية موحدة لكل فرد في سلطنة عمان.ـ دخول موحد لكل الخدمات الحكومية والخاصة بلمسة واحدة ـ أكثر ذكاءً و شمولية من خلال منصة تجمع (بيانات الفرد الحكومية ، وسجله الصحي ،و بياناته التعليمية، و معلوماته البنكية ، وخدماته المجتمعية) تسهل حياة الناس و توفر الوقت و الجهد و تدعم رؤية عمان ٢٠٤٠ في التحول الرقمي و خدمة المجتمع
